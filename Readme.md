# Fake Product Identification System
This project is a web application that utilizes blockchain technology to identify fake products. It allows users to verify the authenticity of products by scanning QR codes or entering product information. The system uses a decentralized ledger to store product data, making it difficult for counterfeiters to manipulate the information.

## Installation

1. Install the Node_Modules
```
npm install
```

2. Build The Project
```
npm run build
```

3. Run The Project
```
npm run start
```
